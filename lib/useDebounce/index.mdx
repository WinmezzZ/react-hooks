# useDebounce

防抖，一般用于输入时防止连续触发请求，减少服务器开销

<CodeBlock>const debounceValue = useDebounce(value, delay);</CodeBlock>

## Props

### value

- 类型: `any`
- 默认: `--`

要防抖的值/状态

### delay

- 类型: `number`
- 默认: `--`

延时的时间，单位为毫秒
